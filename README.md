API de orquestación de seguridad con dashboard visual para análisis y monitoreo de vulnerabilidades en tiempo real. Implementa microservicios en contenedores con buenas prácticas de **hardening** en el despliegue.
- Integración de herramientas externas (VirusTotal, Shodan) mediante APIs
- Priorización automatizada de CVEs con métricas CVSS
- Arquitectura orientada a eventos para alertas en tiempo real

---

### 😴 Sistema de Detección de Somnolencia
**`Python` · `Flask` · `OpenCV` · `JavaScript`**

Motor de **visión por computadora** del lado del servidor integrado con interfaz web responsiva para generar alertas de seguridad en tiempo real. Detecta fatiga mediante análisis de puntos faciales (landmarks) y el algoritmo **EAR (Eye Aspect Ratio)**.

**`Node.js` · `Docker` · `Linux` · `Modelo C4`** · `2026`

Lideré la reestructuración de una aplicación monolítica heredada hacia **microservicios**, documentando la topología de red con el Modelo C4 y asegurando cumplimiento de normativas de arquitectura segura.

---

### 🏪 Sistema Distribuido de Gestión de Tiendas
**`C#` · `.NET Core` · `SQL Server`**

### ⚖️ Juez-Seguro (Secure Judge)
**`SQL` · `Cybersecurity` · `Software Development`**
